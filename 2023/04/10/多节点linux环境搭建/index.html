    多节点linux环境搭建

    发表于 2023-04-10 分类于 linux
    实验环境及软件清单
    • win系统
    • vmware虚拟机
    • centos8
    • xshell7
    • fstp
    目标

    本文将构建3个节点的centos虚拟机，主要目标如下：

    • 实现主机与虚拟机节点之间的通信
    • 实现虚拟机节点之间的通信
    • 实现虚拟机与外网之间的通信

    本文参考教程xxx

    1 vmware安装centos操作系统

    1. centos系统可以在 itellyou下载。

    2. vmware在vmware官网下载，激活码见百度网盘。

    3. 新建虚拟机，选择典型创建。

    4. 稍后安装操作系统。因为直接安装的话后期还要变更操作系统的语言，比较麻烦。

    5. 选择 linux 下的 centos8（其他版本也可以自行选择）

    6. 为第一个虚拟机命名，选择安装位置，不建议安装在 C 盘下。

    7. 为虚拟机分配内存空间。这里分配了40g，其他的默认即可。

    8. 下一步后，点击自定义硬件，根据电脑配置为虚拟机分配cpu和内存。注意，这里要配置一下centos文件位置，如图所示。配置完成后关闭窗口，点击完成即可。

    9. 进入虚拟机，移动光标至上一个选项，回车即可安装。

    10. 选择中文，点击继续

    1. 选择“安装目的地”，不然无法点击开始安装。

    2. 点击完成即可。

    3. 修改时间

    4. 根据需求选择，本文选择带有gui的服务器。若无需操作界面，选择最小安装即可。

    5. 点击开始安装后，即可进入安装界面，需要创建root账户的密码和一个用户，等待安装完成后重启即可。

    2 配置网络信息

    1. 进入centos后，现在还不能上网。在右上角进行有线连接后即可上网。

    2. 进入虚拟机设置界面，将网络适配器变更为桥接模式（后期会出专门来讲这三种模式的区别）。桥接模式下虚拟机可以访问于宿主机在同一网段中的其他设备。

    3. 查看物理机（非虚拟机）的ip，如下图，后期需要将虚拟机与物理机配置在同一网段。

    4. 现在修改虚拟机的网络配置文件，在管理员模式下执行cd /etc/sysconfig/network-scripts/到达目录下。可以看到网卡的名称为ifcfg-ens160。

    5. 使用vim修改该网卡的设置，修改后的配置文件如图所示。

      • 测试与物理机的连通性（物理机ip为172.20.10.2）

      • 测试与外网连通性
